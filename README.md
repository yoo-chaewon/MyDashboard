# 🌃MyDashBoard🌃-ChromApp

#### 소개

**`momentumdash`** 이라는 구글 크롬 익스텐션을 보고 JavaScript를 사용하여 만들어 보았습니다.



#### 주요기능

- 사용자의 위치를 읽어 화면에 날씨 및 기온 출력.
- 현재 시간 출력.
- 사용자의 입력을 입력 받아 저장.
- To Do를 입력 받고, 삭제할 수 있음.
- 새로운 창을 띄워도 사용자의 이름과 할일은 기록 되어 있음.
- 랜덤으로 여러 배경화면 보여줌.



#### 실행방법
PROJECT파일에 index.html 실행(크롬 최적화)
